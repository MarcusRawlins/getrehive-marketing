# ReHive Marketing

Marketing site rebuild for `getrehive.com`.

## Direction

- Reposition ReHive as a custom development studio for small businesses
- Keep the dark + gold palette from the current brand
- Push the visual system toward futuristic, upscale, motion-led storytelling
- Build the site around SEO-driven service and vertical landing pages

## Current scope

- New Next.js App Router marketing site
- Homepage foundation with premium motion and contact flow
- Technical SEO baseline: metadata, schema, robots, sitemap
- Research notes in [docs/seo-research.md](./docs/seo-research.md)

## Local development

```bash
npm install
npm run dev
```
